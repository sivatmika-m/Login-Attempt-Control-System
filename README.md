Login Attempt Control System
Objective

The Login Attempt Control System is a Python program that limits repeated incorrect password attempts and restricts access after a fixed number of failures.

Features
User authentication using password verification
Limits the number of login attempts
Restricts access after multiple failed attempts
Demonstrates basic security mechanisms
Technologies Used
Python 3
How It Works
The user enters a password.
The system checks whether it matches the correct password.
If the password is incorrect, the attempt counter increases.
After three failed attempts, access is restricted.
If the correct password is entered, login is successful.
